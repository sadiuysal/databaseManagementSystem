Project 
----------------------------------
Designing an example DBMS and implementing it in php
 
----------------------------------

Prerequisites
----------------------------------
-See ER_Diagram for understanding DB the structure.
-Download XAMPP for php files.
-See good tutorial for this project: https://www.youtube.com/watch?v=jp4FVwmZMGQ&t=356s
 -Put project folder to XAMPP system folders(generally /opt/lampp/htdocs/)
Running
----------------------------------
-Start mySQLDatabase and Apache Web Server from XAMPP
-You can start by creating a DB, go to : 
http://localhost/projectFolderName_in_xamppFolders/createdb.php
-Then you can select admin or user by typing it.
-Depending on the user type you can start doing database operations with corresponding buttons.
-Go to http://localhost/phpmyadmin/index.php for checking operations you have done in DB.
